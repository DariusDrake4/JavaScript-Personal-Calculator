# JavaScript Personal Calculator

A simple web-based calculator built using **HTML, CSS, and vanilla JavaScript**. This application performs standard arithmetic operations through an interactive button-based interface.

## Overview

This project was created to practice core JavaScript fundamentals, including DOM manipulation, event handling, and basic logic control. The calculator allows users to input numbers and operators using on-screen buttons and displays results dynamically.

## Features

- Addition, subtraction, multiplication, and division
- Clear and responsive calculator display
- Button-based user input
- Real-time calculation updates
- Simple and clean UI

## Technologies Used

- HTML
- CSS
- JavaScript (Vanilla)

## What I Learned

- Handling user input with event listeners
- Updating the DOM dynamically
- Managing application state in JavaScript
- Writing clean and readable front-end logic

## How to Run

1. Download or clone the repository
2. Open the `index.html` file in any modern web browser
3. Start using the calculator

## Future Improvements

- Keyboard input support
- Improved styling and layout
- Error handling for invalid inputs
- Advanced operations (percentages, decimals, etc.)
